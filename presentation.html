Post-Session Agent (Automated)

Runs after the consultation ends. An agentic loop using Claude Opus that autonomously generates all clinical documentation.

Autonomous Pipeline
1
Visit Record — complaints, symptoms, diagnoses
2
Session Summary — key findings, follow-ups
3
SOAP Note — Subjective / Objective / Assessment / Plan
4
EHR Entry — ICD-10 codes, orders, prescriptions
5
Clinical Letter — referral or follow-up letter
6
Patient Record Update — merge new conditions
How It Works
Claude Opus receives the full transcript + patient context as input. It calls tools in sequence via an agentic loop — each tool writes to the database. The doctor sees all documents ready for review.

Human-in-the-Loop
AI generates all clinical documents automatically, but every document passes through the doctor for review before being finalized. The doctor can edit any section.
Temporal Context
EHR data is serialized with temporal markers so the AI distinguishes pre-existing conditions from newly reported symptoms — preventing false extraction.
